有三五个过来秀智商说我卖课的，这完全就是造谣诽谤，很明显根本就没怎么看内容，我就回复让他们指出到底哪里在卖课；后来发现有几个评论不见了，不知是平台还是他们自己给删掉了。

  4. 详聊为何 Web2 终是末路；
  5. Web 前端转型需要掌握哪些知识与技能，以及什么样的人不适合转型；
  6. 初阶教学及实战详解；
  7. ……
