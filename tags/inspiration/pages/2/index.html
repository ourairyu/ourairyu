刚在开车的一个来小时里，一口气跟🐷说了三件事（基本都是我在逼逼，她简单地回应）——

第一件是告诉她这几个月因项目忙没时间专注去做的想深入学习研究的几个方面：「领域（domain）」相关的知识、方法论、技术；复杂系统相关的，如复杂性科学、混沌理论；大脑活动相关的，如脑科学、神经科学、心理学、认知科学；知识图谱相关的。

第二件是我初步认为将生活程序化/公式化可以大大降低精神、精力损耗，提高效率。虽然无趣，但很实在。人的情感是很复杂、耗时、耗力的，但生活中大多数为人处世不需要情感。

第三件是不同类别的事物间进行 communication（沟通、交流、通信）一定要有一个通用的桥梁/媒介，如跨国交流的英语、国内跨地区交流的标准语、学外语所用的国际音标、跨程序语言的 API 定义等等。感悟Life-hacking

在去年写的这篇文章中，提到了能够对自己的人生进行「元编程」的「元认知」和「元能力」。

其中，「元认知」可以认为是与「正念」密切相关的（是不是划等号我不确定）；通过了解更多知识后，在这里对「元能力」中的「学习能力」做点简短补充——

目前来说，我认为作为高效学习的前提，有三个条件：

  1. 好奇心、兴趣。俗话说，「兴趣是最好的老师」，好奇心、兴趣能够让你学习东西事半功倍，学习不感兴趣的东西是学不进去的，「强扭的瓜不甜」。
  2. 掌握些脑科学、神经科学、心理学的知识，了解大脑是如何运作的，比如海马体、杏仁核、主观意识、潜意识等。只有了解大脑，才能知道如何科学用脑。注意让大脑得到充分休息，避免早日记忆力衰退、老年痴呆。
  3. 保持练习的心态并持续刻意练习。感悟Life-hacking
