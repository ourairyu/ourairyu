今天初次了解（还算不上学习）泰语，单从文字构成来看，比韩语复杂得多——共 44/42 个辅音/子音，和 32 个元音/母音，以及 5 个声调。

其中，辅音和元音的相对位置并非固定的，有一定的规则（具体暂未了解），并且有的发音相同或很接近，更增加复杂度。

但是，它不像英语和日韩语那样会根据时态等进行变形，而是像汉语一般由不同相对独立的语义单位任意组合，这样更为「亲切」。

泰语中的很多词汇是外来的，如梵语、巴利语、汉语等。

由于泰语的发音特点，想听起来很爷们儿是几乎不可能的……😂😂😂外语

这几天写了两篇自己向 Web3 转型的经历总结文章，除了个人网站，还同步发到了公众号、掘金、SegmentFault 等平台上。

由于我在掘金的创作等级正巧刚升到 LV.4，有了自动推荐到首页的权益，大大提高了曝光度、互动率和涨粉速度。

评论留言的人中绝大部分在说 Web3 割韭菜、犯法、坐牢什么的，这些人我都没回复。

有三五个过来秀智商说我卖课的，这完全就是造谣诽谤，很明显根本就没怎么看内容，我就回复让他们指出到底哪里在卖课；后来发现有几个评论不见了，不知是平台还是他们自己给删掉了。

当然，应该是有更多的虽未发声但智商在线的掘友的，因为我收到了数量超过评论的点赞和收藏通知。

由此可见，Web3 被妖魔化得比较严重，需要有更多真正在做事的人加入到为 Web3 正名并传播正确观念的行列当中——我要成为一分子！

我在训练营学习时发现，能让 Web 前端从 Web2 平滑过渡到 Web3 的中文体系化资料很难找到，我觉得这是自己比较擅长的事，可以先写几篇试试水，让我感受到需要的人多了，就继续多写，否则可能就偶尔写下。

鉴于此，我在公众号上建了个合集，并把阐述对 Web2 的失望及向 Web3 转型经历的文章加入其中，整体大概是这样的叙事脉络：

  1. 浅谈我为何觉得 Web2 终是末路；
  2. 总结我找到 Web3 这个出路并转型的经历；
  3. 厘清跟 Web3 相关的几个核心概念，为其正名；
  4. 详聊为何 Web2 终是末路；
  5. Web 前端转型需要掌握哪些知识与技能，以及什么样的人不适合转型；
  6. 初阶教学及实战详解；
  7. ……

虽说我把这个合集挂在了公众号的「付费合集」下面，但绝大部分文章是免费的，只有技术剖析、实战详解类文章会按单篇付费的形式设置——为了推广普及 Web3，只能如此……
