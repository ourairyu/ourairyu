「反混沌」（Anti-chaos，a.k.a. NTKS）体系在工具方面目前主要有四条线——

「Petals」提供原子 UI 组件，可一定程度上定制，每个 UI 组件是一个包，核心与 React、Vue 之类的无关，任何人都可以基于核心选择自己渲染结构的方式。

「Buds」提供 UI 组件集成工具与服务，与「Petals」的理念结合，让「组件库」只限于个人或组织自己。

「Organik」是一个视图库无关的应用框架，像有机体一样。

「Handie」则是融合了上面每个的让各个技术栈（主要指视图库）在开发中后台应用时拥有几乎一致体验的开发框架。反混沌PetalsBudsOrganikHandie
