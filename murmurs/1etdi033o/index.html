欧雷
发表于 2021-02-01 01:06

昨天在群里讨论到一种现象——人们总是只知道自己所知道的，对已经存在的自己不知道的部分一无所知，然后总会「重新发现」。

这与我之前说的「历史总是相似的」实际上是一回事，可分为四个方面——

  1. 信息的存储方式；
  2. 信息可传递并尽可能在传递过程中减少流失；
  3. 信息检索与接收方式；
  4. 某个人去寻找信息。

前三个是整体需要解决的，最后一个是个体问题。

对于个体来说，看书是增加知识的一种方式，思考是增长智慧的一个途径，但有时需要更高效、更精准地临时找到某个知识，现有的网络环境和手段暂时无法很好地满足这一需求。

如果 Web 存储了所有有效知识并且可访问（不会失效，没有管制等），人的大脑直接与 Web 相连，某个装置将意识直接转化为精确的想要搜索的知识描述，搜索结果也能直接转化为让大脑接收、理解的东西，这无疑是种非常高效的方式。

这会涉及到本体工程、语义网、AI、脑机接口、神经科学等等，我觉得是个值得探索的方向。

历史总是相似的，为什么前人犯过的错，后人不吸取教训再去犯—— 前人的事件、知识没有记录； 前人的事件、知识有记录； 记录遗失； 记录顺利流传下来； 后人在解读时意思与前人所表达的不符； 因为前人的记录总和可以看作是无限，后人无法全部了解并吸收； ...查看详情
