欧雷宇宙

随着这些年向内对自我的探索，精神世界越发成形且体系化，便想为之起个「吊炸天」名字——欧雷宇宙（Ouraiverse）；互联网上的我是「我」本身一部分的展现与延伸，鉴于与我精神世界的相似性与关联性，将其也叫做「欧雷宇宙（Ouraiverse）」。

动机

Web 的初心是开放、去中心化的，而随着建立在 Web 之上的商业的发展，逐渐涌现出众多（试图）掌握每个人命脉的中心化（内容）平台——不仅为它们免费提供内容，自己生产的内容还可能会任其清除。

内容是自己的智力劳动成果，其所有权、主动权等理应掌控在自己手中！本计划是符合 Web 初心的，算是我在个人知识库体系这方面理念与理论的实践。

蓝图

第一阶段

将各类数据从外部平台上转移到本地集中式地中心化管理，形成个人知识库。

数据存储

按照 QiiDB 的数据规范将各类数据存储为本地文件并同步到远程 Git 仓库。

数据转换

编写数据批处理脚本将个人知识库中的数据转换为静态网站生成器所需格式。

第二阶段

与知识呈现相关基础能力的建设。

作品展示

我会通过图文、音频、视频和代码等方式进行创作，由于开源软件相对特殊，需要额外专门的站点，其他类型的在本站上展示。

知识共享

另设独立站点支持本地个人知识库内容通过公开或受限方式共享。

产品运营

正如上文所述，会用多个独立站点去切割通过不同形式与方式共享的本地个人知识库内容，为方便个人品牌的建设与运营，得有专门用来介绍各站点并记录它们发展变化的网站。

个人名片

站长、作者等的介绍信息维护在 LinXoid 上并链接到个人页，详见相关计划的「互联网名片」部分。

第三阶段

知识管理与输入相关基础能力的建设。

外部平台

打造并深度使用去中心化（去平台化）的个人知识库体系不代表与其他平台之间相隔绝，根据运营策略会适当地选择合适的方式去与其他平台相结合。

能力互补

有些功能单凭个人网站无法胜任，需要借助外部平台来弥补短板。

视频作品

将哔哩哔哩（B 站）作为 vlog、聊天节目、知识分享、视频教程等以视频为媒介的作品的主要发布平台，根据定位和内容类别细分为以下几个频道：

在研发出自己的移动客户端之前，仍像往常一样用常规方式在朋友圈发信息，但在有自研客户端后就通过内置的分享功能分发到微信。
