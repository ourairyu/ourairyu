我的博客简史

欧雷 发表于 2021-05-28 23:16:09

0 条评论

标签：博客豆瓣化

这是一篇承上启下的文章——回顾写博十多年的心路与博客的发展历程，并披露网站接下来的发展方向。

写文的原因

我从小就喜欢去收集信息并整理出来形成小册子，像记录 FC 游戏的出招、密码等的「秘笈」，还有《塞尔达传说》等游戏的攻略——

除了信息，从小到大我还收集并收藏了一些类别的实物，如：陈小春与周杰伦的录音带；Berryz工房、℃-ute、AKB48、渡り廊下走り隊、VOCALOID 等日本歌手的 CD 光盘；《子不语》、《长歌行》、《陽当たり良好！》等中日漫画；FC、GB（C）、GBA、3DS、Switch 等的实体游戏；邮票；钞票与硬币。

可以说，「收集」并「整理」是我最大的癖好，这会使我有很强的充实感和掌控感，也是促使我写文章和想要出书的「本能」。在「本能」之上，「写文章」这件事也包含着两个主观上的「目的」——

将某一时刻或某段时期的思想、感受通过文字的形式「持久化」，因为人的记忆和情感都是暂时的，活性会随着时间的推移而降低甚至消失。时隔多年之后，可以通过曾经留下的文字「读取」进脑中，尝试重建当时的情景，回味并对比下与当前的状况有何不同，重新审视下自己。

自己所写的文章，只要不是私密的，就可能会被其他人看到。如果文章的内容不是单纯的流水账与情绪宣泄，而是带有一定的思考或经验的总结在其中的话，总会在某个时刻帮助上其他人的——你所遇到的问题和烦恼，不只是你一个人会遇到——分享思考或经验的总结，就是第二个目的。

文章的变化

虽然小学就去过电脑房了，但我开始接触网络是在初二、初三时，当时正火的网游是《传奇》和《梦幻西游》。那时我还不怎么上网，去网吧也就是聊聊 QQ 或跟同学联机打 CS。高中时上网的话，除了在 QQ 聊天，再就玩玩网游，看看动漫，逛逛论坛。我是上了大学才开始在网上写文的。

现在回过头来看，从我开始在网络上写文时起，这十多年来我的文章内容的变化大概可以分为「萌芽期」、「半熟期」和「成熟期」这三个阶段——

萌芽期

起先，我所写的文章是日记型的，每篇都是校园或日常生活中比较触动我的事情，写写自己的心情感受和一些小感悟。由于是发表在 QQ 空间或校内网上的，好友基本都是熟人，他们有些人看到后或因感同身受或想安慰鼓励而留下评论，视情况我也会回复。

在这期间，我觉得只是写写「日记」也未免太「普通」了点，就想到：「何不把这些日常生活经历整理成一部小说呢？」于是，以第三人称视角叙述的半自传体小说《欧雷物语》诞生了。遗憾的是，这部小说只写了几个篇章，并且没有公开发表。

半熟期

当我离开校园成为众多社畜中的一员后，在社会的毒打和职业的熏陶之下，我想说的话变得更多，可写的范围变得更广，对文章内容的措辞、结构和节奏也掌控得更好，一篇文章一不留意就一、两千字了。

从这阶段开始基本不写日记型的文章了，以思考并总结平时所观察到的现象为主，内容变得更为宽泛、抽象；其次是与职业相关的，这方面的文章几乎都是且只能是一些「招式」，当时的 level 在那呢，写不出什么「内功心法」；再者就是日语学习和游记类的。

成熟期

「萌芽期」和「半熟期」的文章，更倾向于单纯的记录，利己的因素占据了绝大部分；而「成熟期」的文章更侧重于输出体系化的知识、方法论，在梳理自己想法的同时更希望能够帮助到他人，放大文章的价值。

这阶段所写的文章主要以系列文的形式体现，以便于日后想要出书时方便厘清脉络和整理编撰。

博客的变化

我一开始是在 QQ 空间或校内网上发表文章的，用过的人都知道，它们的 UI 设计就像学校的校服一样——很 low！虽然可以换皮肤，但在不付费的情况下可选项很少，并且都很土！

追求个性，不愿随大流的我当然不可能一直忍受「跟别人差不多」，在逐渐远离校园生活时我也开始寻找可以在网上展示自我的那一片天空。

寄人篱下

2009 年春天中的某几天，借助于搜索引擎在广阔无垠的信息海洋中查找可以自定义外观的博客平台。记得最先找到的看似符合要求的是百度空间，但稍微试用下发现还是不满意。左挑右选，辗转之下，最终选了博客大巴。

既然落脚点已经找好，该到给自己的博客起名字的环节了。起名字或者说命名向来是一件很难熬的事儿，尤其是这种起了之后不能轻易改的情况……思来想去，最后决定用「宅男的部屋」作为博客名。

「宅男的部屋」中的「宅男」不是大众所认知的那样——

不知从何时起，「宅」成了「闭门不出」、「家里蹲」的代名词，作为「御宅族」的我对此感到十分厌恶！由于「宅文化」在国内没有被正确传播并且没那么普及，大众普遍对它没有正确的认识，所以才会曲解「宅」的含义；再加上国内对「宅文化」所涵盖的主要内容本身就存在一定歧视，导致「御宅族」注定是个小众群体。

那么，「宅」是什么？「宅文化」包含什么？

一般来说，「御宅族」是指那些对「ACG」（即动画、漫画、游戏及相关的人事物）或「idol」（通常是日本偶像）痴迷的狂热爱好者，前者叫「动漫宅」，后者为「偶像宅」。在「御宅族」之间，「宅」可以理解为狂热程度，说一个人很「宅」，就代表他很狂热。正因为「御宅族」都具备「对某事物、某人痴迷狂热」的特点，喜欢除「ACG」和「idol」之外的狂热爱好者也可称为「御宅族」，如：「铁道宅」、「军事宅」等等——「宅」是热情！「宅」是一种态度！当然，并不是说一个人光热爱某事物就能被叫做「御宅族」！「宅」是一种精神，「宅」是一种信仰，其中内涵需要用心体会！

「宅文化」所包含的内容主要有：动画、漫画、游戏、轻小说、VOCALOID、偶像与相关的角色、声优（配音演员）、cosplay、插画、同人作品、展会及其他各种周边。

「宅主义」计划

而「部屋（へや）」在日语中是「房间」的意思——「宅男的部屋」的字面意思是「御宅族的房间」，稍微内涵点，就是「我的天地」了。顺带一提，日本御宅族的房间大概是这样的——

接下来就该按照自己的想法给「宅男的部屋」进行「装修」了，在「白色的墙面」装满自己精心设计的「小部件」。

由于当时我被选中参与校企合作的项目，脱离了常规的校园生活，像实习生一样每天过着打卡上、下班的日子。在开始鼓捣博客的那段时间我白天正常上班，下班后吃完晚饭就回到公司「加班」。

小小的办公室里没有别人，灯也不用开了，好好享受只有一个人的安静黑夜。正是那段时间，我知道并喜欢上了初音未来和 supercell，每当我真觉得孤单空虚寂寞冷了就听 TA 们的歌。

经过多个黑夜，我一边在欣赏别人博客的同时想象着自己博客该弄成啥样，一边在网上找 HTML 和 CSS 的教程学习怎么把博客弄成自己想的那样。功夫不负有心人，在经历一番折腾后终于出了点成果——能看了，并且浏览器兼容性还行——在多个网吧通宵折磨出来的……

博客的外观搞好后就要填充内容了。除了写些文章之外，还加入了 ACG、idol 相关的资料，可以说「宅男的部屋」越来越名副其实了。

因为那时我很喜欢 supercell，想弄个粉丝网站，就仿照官方网站的样子在博客大巴上又弄了个站点，记忆中是叫「supercell Fan Site」。我将官网的信息翻译成中文发到粉丝网站上，并因此认识了一些同样喜欢 supercell 的网友。

自立门户

把内容发布在别人家的平台上，难免会被各种审核从而导致内容发不出去，写个字说个话还要提心吊胆的，生怕哪个字用错了就被毙掉。追求自由的我讨厌被束缚，会想方设法挣脱让我觉得难受的枷锁——

我是一个自我意识非常强烈的人，因此我叛逆、独立，有想法，追求自由，强调自己的独特性。为了自由，我可以舍弃一切，冲破任何束缚我的阻碍，哪怕是社会道德、骨肉亲情、凡尘人世。

欧雷《去你的「正常」人生！》

在博客大巴过了两年多「寄生」生活后，于 2012 年初我「搬」到了自己购买的虚拟主机上。做出这个决定在当时也是有些纠结的，因为每年都要给主机和域名续费，多出一笔固定花销。

从个人品牌塑造角度来说，域名越短越易记越好，我就用自己的英文名加上 .ws 这个看起来有点特别的顶级域名组成 ourai.ws 作为网站域名；并将网站名字改为「欧雷流」，配上标语「不走寻常路」，这个搭配更好地表达了我想表达的——

「欧雷」就是我，替换代入为「我流（がりゅう）」，意为：自成一派；独树一帜；别具一格。因此，广义上的「欧雷流」是具备我的个人色彩的事物的集合：思想、言行、穿着、做事方式等；狭义上则专指我的思想体系。

欧雷《欧雷与欧雷流》

我好像一直都不满足于自己的网站只写写文章，总是想再加些额外内容，尤其是离开博客大巴后自由度更高了，由于用的是 WordPress，除了前端代码还能再在后端搞点事情。

不仅是文章，连 ACG 相关资料我也「搬」了过来，都存到了 MySQL 数据库中。当时还基于 Flickr 提供的 API 做了相册功能，但后来好像是因为不再对外开放或是网络不通了的原因把这个功能下掉了。

我一直想搞个像 SNS 时间线的功能，用来展示我的活动轨迹，但鉴于各种原因，这个功能没有很好地实现出来，只显示文章相关数据糊弄了事儿——

过了三年左右，也就是 2015 年，有些厌倦了 WordPress，不想继续使用 MySQL 数据库来存我的内容，于是我又进行了一次大迁移——开始使用 Jekyll 这个静态网站生成器，并利用 GitHub Pages 托管生成后的站点。

这还没坚持到一年就又「搬家」了……这次是因为百度无法收录托管在 GitHub Pages 上的网页了，再考虑到未来很可能会用 Node.js 写点服务，就买了阿里云 ECS，这下每年的固定花销又涨了好几百，肉疼……

在迁到阿里云之前，我的网站风格进行了一次大升级，设计时以提高阅读体验和保持简洁风格为最高设计原则，体现个性在其次——

这包含几个小彩蛋的设计一直沿用到现在，已经五年多了。

展翅翱翔

可以说，在迁到阿里云之后，我的自由度已经很高了，能够比较尽情地去实现自己的想法了！

得益于这些年业界技术的发展以及自己见识的增长，在去年夏天我筹划了「开放数据」计划并开始对网站进行「豆瓣化」改造，陆续加入了「想法」、「我的」模块，一直想做的 SNS 时间线那种功能的「时刻」也完成了——

在接下来这可见的三年内，我的网站在风格上依然不会有什么大变化，重点要做的是「豆瓣化」和模块化改造。

做这些为什么要三年这么久？最大的原因是这些是在业余时间做，并且要做的事情还是挺多的，实际时间没准儿还会超过三年。

结语

很多人说博客已经过时了，我却不这么认为，反而会越来越火——追求个性和自由的人越来越多，博客是非常好的展示自我的舞台！

欧雷流——不走寻常路！
