我若出书，你欲看否？

欧雷 发表于 2021-01-16 11:05:23

0 条评论

看到标题，估计有人会想：「啥？你想出书？！你怕不是来搞笑的吧？你何德何能？！」——是的，我不是来搞笑的。

请看我这正儿八经一副严肃地要吃人的脸，像是在开玩笑吗？啊？？？

出书原因

我已不止一次跟别人说过「我想出书」这件事，有此想法，既不为名亦不为财，主要是有以下三个原因——

做有价值的事情

就像上周六发布的第一期自说自话的谈话节目中所说的那样——

希望做些有价值的事情，人活着总得做点什么东西影响、帮助到更多的人；如果可以的话，帮助社会或帮助人类——这是一个人活着最大的价值，我认为。

欧雷《欧雷说：「介个银居然又给自己挖坑了！」》

「有价值」意味着「会帮助到别人」，哪怕是仅有一两个人。如果能对社会或人类产生正面影响，那就再好不过的了！不过，我几斤几两自己清楚，不敢奢望自己做的事情能够影响社会或人类，只求尽可能帮助到更多的人。

我已经积累了好几个自认为「有价值」的事情，如（按记忆中想法出现的先后次序排序）：宅主义、Life-hacking、前端知识与技能考核评价体系（FES）、反混沌、以个人为中心的服务等。

「Life-hacking」、「以个人为中心的服务」和「反混沌」是我认为最有价值且最可能出成果的三个。其中，「Life-hacking」与「反混沌」跟本文主题「出书」有关。

所思所学体系化

「所思」是指我的思想体系——欧雷流。上面说的「Life-hacking」是其一部分，并且是重要组成部分！

我自称为「生活黑客」，即「Life hacker」，是掌握了「Life-hacking」并能熟练运用的人（这是未来完成态，现在还在研究阶段）。

关于「生活黑客」这个词，网上给的解释大多是那种用一些让生活变得更高效之类的技巧，但在我这里它有着更为深度的含义——能够改变生活、玩转人生的人。而「Life-hacking」就是让人成为「生活黑客」的一套理论体系——

一个人能够做成一件事，是因为他找到并掌握了方法——假设世上任何事情都能在允许一定误差的条件下总结出一套通用的方法，那么「人生」这件很宏大的事情也应该同理。

欧雷《将人生置于股掌》

「所学」则是我的专业与职业相关的知识、经验总结。

让我活得久一点

人的肉身毁灭并不代表他真的死了，只要有人记得他，他就是「活着」——有的人活着，他已经死了；有的人死了，他还活着。

浙江这边的人挺长寿的，80 多岁的很多，活到 90 以上甚至上百的也不少。我觉得自己可能活不了那么久，并且不想自己处于生活不能自理或者老年痴呆之类的状态；这种情形对我来说已经不算是「活着」了，我宁愿选择自尽或者安乐死。

正因如此，我想至少让我的思想、精神能够活得更久些，代替肉身的我去帮助别人。于我而言，最切实可行的就是创造尽可能经久不衰的作品，比如书和软件。

为什么博客不行？

几个月前我就在担心——万一哪天我死了，我的那些数字资产该咋办？我所能想到的，就是在那之前把我的个人网站的源码全部开源在各个 Git 托管商上，然后选择一个活得能够相对久一点的平台部署上去；如果是需要花钱的云主机，就买好几年的服务，域名也是。

我还是觉得实体书可能会存在更久一些。

书的内容

稍微关注我的人都知道，我已经开了好几个文章系列了，有的写了一篇，有的写了几篇，有的还没开始写。可以说，这些系列化的文章就是为了日后出书作准备，文章系列也是我有了出书的想法之后才产生的。

我要出的书，根据内容分，有两类——

关于世界的

述说我观察到的、我所理解的、我想象中的、我理想中的世界是什么样的——欧雷的世界——这是书的主题。这书会涉及到哲学、科学、工程学等方面的知识，也会有很大一部分我的想象力的产物。

想了解世界并不单单是对宇宙万物充满兴趣，同时也是为了沉淀掌控并玩转人生的理论，也就是上文中的「Life-hacking」。

在作为（我所写的）此类文章的第一篇《人生元编程》中所说的「元能力」，和前不久刚发的年总结文章《属于我的三年·第一年》中提到的「元模型」，都是「Life-hacking」的一部分。

半年前我开了一个关于世界的文章系列——重新定义「世界」。目前仅写了第一篇，《客观的现实世界》。这篇文章中并没有什么高深的知识，只是列举了几个比较浅显的，大部分人能够看懂的，并且我认为是偏底层的概念——形式要义（客观存在与主观意识）、实体关系（「世界」和「人」之间的关系）、系统理论（一般系统与系统思维）。

理所当然，这个系列的文章肯定不止那一篇——

因此，我要写的关于世界的书也不是一本，而是系列作，是《重新定义「世界」》文章系列的补充。

当然，如果可能的话，我想把现实世界、数字世界、精神世界和融合世界都写了，但这不太现实，能把前三个世界写完组成「三部曲」我都谢天谢地了……写完前两个世界是比较现实的，至少要把现实世界写完，这是我的底线——这部分就是为了阐述我的「Life-hacking」理论！

我连书的封面都想好了——

就像上图中那个侧面解剖的发音部位图一样，手绘一个侧面解剖的头部，大脑部分留空，根据不同的世界里面填充上能代表那个世界的符号。

关于软件的

内容主要是以前端工程师视角看待软件设计、软件工程，以及职业相关问题的思考、经验总结。

鉴于与上文所说的现实世界相关的内容相比，这部分的重要性相对低些，所以就先不展开说了。

如何实施

由于职业等因素，我不可能集中一段时间去专门写书，因此会先在平时以文章的形式保持输出，也是作为零散的记录存在。等我认为时机到了，再将零散的文章编撰成书。这个「时机」，可能会五年、十年不等。

书的形式，肯定会有并且是先有电子书。「世界」相关的电子书，在没有出版社联系我出实体书时会一直免费，如果有出版社联系我出版实体书的话，可能会因为他们那边的原因而对电子书进行收费；「软件」相关的电子书会根据内容而决定是否收费，免费的如果有出版社联系我出实体书的话，处理方式与「世界」相关的电子书相同。

「世界」相关的书，就算没有出版社联系我出版实体书，我也会自掏腰包出版，所以这会是限量版。到时我会拿出几本赠予亲朋好友，剩余的通过什么活动的形式免费给那些长期关注我的人，以示感谢。

最后——

如果我出书了，你想看吗？
