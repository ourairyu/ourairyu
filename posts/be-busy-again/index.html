不躺了，我要认真做创作！

欧雷 发表于 2022-08-04 08:59:19

0 条评论

标签：Life-hacking反混沌

遇事不决，量子力学。

在所长林超的一期视频中，提到了「躺卷二象性（sleep-fold duality）」这个很有意思的词——

这个特性就是卷不赢，躺不平——想站成 90 度，发现太卷，放眼一看，好机会全都轮不上自己；想躺成 0 度，发现太穷，掐指一算，财务自由依然遥遥无期；结果只能活成 45 度，但又发现保持这个姿势它也不容易。

所长林超《卷不赢，躺不平，怎么破？》

当前的社会环境不好（好过吗？），绝大多数的男女老少都处于想躺平却又无奈起来卷的境况。

是躺？还是卷？

关于「工作与生活平衡」这个绝大部分人都会面临并烦恼的问题，可以说迄今为止我已经经历了三个阶段——

今年 4、5 月份之前，说实在的，对于工作和生活到底应该怎样依然没有刻骨铭心的那种深刻感——工作上阶段性打满鸡血，间歇性心理低谷；生活上无法完全放松，娱乐时有些许罪恶感，时常感到焦虑、有压力——步入社会后一直如此。

在 4、5 月份遇到一些事情，结合以往的亲身经历，对人生的根本性问题进行一番思考后幡然醒悟——

活着就是健康地等待生命的完结，一切皆虚无，没什么事是有意义有价值的，不要寻找意义，别想创造价值。

欧雷《向过去的自己告别》

在那之后，我就从心态上躺平了，听从内心的声音，不想工作而想要看乱七八糟视频时，完全没有心理压力了——什么是「焦虑」？「精神内耗」是啥？

既然心态上躺平了，除了为解决工作使用中遇到的问题之外这几个月在不上班时就没怎么写过代码，「反混沌」体系几乎没任何进展。

相反地，我通过各种渠道，如公司「内网」、小破站、微信等，去了解信息进而观察人类行为，思考总结出一些理论并加以验证。

这期间我在小破站上看的与人类起源、神话传说、诸子百家、神秘组织、宗教、历史、物理、天文、生活记录等相关的大量视频，大大补充且完善了我的三观。我并非刻意地强迫自己去看那些视频，而是那时那刻内心想要去看，我只是顺从而已。

然而现在忽然又意识到——我只是心态上躺平了，肉体上暂时还没有且无法躺平；在无老可啃的前提下，只要不到寺庙当和尚或去道观做道士，就得为五斗米折腰，不得不考虑现实的生存问题——还是得打鸡血地工作，还是得感到一丝压力。

上述三个阶段，我觉得可以总结为——

参禅之初，看山是山，看水是水；禅有悟时，看山不是山，看水不是水；禅中彻悟，看山仍然是山，看水仍然是水。

青原行思

躺不下去，那就起来继续卷咯！

怎么卷？

大多情况下，工作是反人性的事情，有很多令自己不爽的地方；若是不得不面对，只能想方设法减少让自己不快的因素。

很多人对做事的观念有个严重的误区——认为只要努力、奋斗就行了，终会有回报。

首先，在继续朝着某事它本身所指向的方向发展之前，先质疑并论证这事本身及其前提的合理性、正确性，若是不合理的、错误的，努力、奋斗就是在做无用功，不仅白费力气，还可能加重事情/事态的混乱程度。

其次，不要用蛮力，而去使巧劲儿，用四两以拨千斤，也就是寻找「杠杆解」。

那么，在「工作」这种事上，所谓的「杠杆解」是什么呢？细枝末节暂且不论，在大方向上——从事自己喜欢并适合且处于朝阳期的行业/领域，在此基础上采用成本尽可能小而收益尽可能大的途径。

对我来说，这个「杠杆解」应该就是主要通过数字化手段将知识与能力变现的创作了吧？

之前已在文章中或其他途径说过多次，并且一直以来较为坚持地在做，只不过此时此刻我内心的想法更加清晰、确信、坚定，想在此更为详细地描述出来——

创作题材

我的创作聚焦于「欧雷流人生工程」和「反混沌前端工程」这两个题材，内容的形式及范围为：

  • 图文——以体系化的思想理论为主，时机成熟可整理并发布/出版为电子书、纸质书；
  • 音频——针对某个话题在播客中进行杂谈或访谈；
  • 视频——日常、出行、旅游等生活记录，以及技术分享、教程讲解等。

这几种形式对「欧雷流人生工程」很是重要，于「反混沌前端工程」而言，提供企业级 Web 开发解决方案则是重点：

  • Fxxk Design——以「设计即代码」为目标的跨 JS-based 技术栈、跨平台 GUI 开发体系；
  • Future.js——以「需求即代码」为目标，将前端应用各个层次、各个环节之间的通信规范化、标准化的开发体系。

盈利方式

在积累阶段，我的绝大部分创作是免费的、开放的，凭看客心情通过支付宝、微信支付等途径进行赞助/打赏。

但产出优质内容是需要投入很多时间和精力的，光靠爱发电是行不通的，无法持续优质创作，届时会采取一些手段变现，如：

  • 对内容质量分级，相对较高的需要付费，可以是会员/粉丝制；
  • 基于「反混沌」体系的定制化服务；
  • 人生工程、软件工程（主要是 Web 开发）或人件工程相关的咨询服务。

除了这些，还可接外包或其他形式的合作。

持续发展

写本文的原因之一就是想郑重地声明一下，上面提到的事情我是认真的，真正当作事业去做的，我所提供的 Web 开发解决方案不会无缘无故地就鸽掉，若是能解决遇到的问题就请放心使用。

不过也说了，这些事想要持续且良好地发展下去，光凭我自己的爱和热情是不足以维持的，需将其转化为实实在在的经济来源才行。

故此，若喜欢我的创作，或提供的方案确切解决了问题，还请为我的劳动成果进行酬劳——消费者的经济支持与提供者的认真负责是相辅相成的。

除了经济上的支撑，这么多事情光靠我一人显然不行，所以也会在人力上寻求帮助。

第一个自然是身边最亲近的人——俺家🐷！她主要是在内容创作方面帮助我，包括中英互译、音视频后期、推广运营等。

在 Web 开发解决方案的代码开发方面，就是以开源项目/组织的方式运作，当像 Vue.js 一样有比较长期、稳定的铂金、黄金赞助商之后，可以考虑付薪找远程全职开发者。

总结

很大程度上，卷是不可避免的，就看是在哪个维度卷，用什么姿势去卷，要卷成什么形状。
