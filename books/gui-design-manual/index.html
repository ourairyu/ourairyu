给前端开发者看的 GUI 手册

欧雷 著

0 条评论

本书旨在总结一些在做页面、应用开发中会用到的控件的展现与交互形式，并作为一些人在去开发相关控件时的参考。促成本书的经验虽来自 web 前端，但其价值并不局限于 web 前端。

为什么会有此书？是因为我一直想找个罗列了各种控件的定义及其交互逻辑的文档或手册，然而并没找到。如果我的这本是世界上或中文圈的第一本，那么我会觉得很荣幸，能够为帮助到他人及为了标准化贡献微薄之力！
